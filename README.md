# X Perspective Switcher

> 一个 Chrome 插件，让你在 X/Twitter 上一键从不同视角生成独特评论。

💰 穷人视角 | 🐴 牛马视角 | 📚 学生视角 | 👔 领导视角 | 📰 记者视角 | 📈 投资人视角 | 🇺🇸 美国人视角 | 🧐 杠精视角 | 👩‍🦳 老妈视角 | 💅 毒舌博主 | 🐱 猫奴视角

## ✨ 功能特点

- 🎭 **12种视角一键生成** — 从穷人到投资人，从杠精到猫奴，每个视角都有独特人设
- 🔄 **3种生成模式** — 反向解读 / 直接回复 / 相似话题
- 🤖 **多模型支持** — 硅基流动（免费）/ OpenAI / DeepSeek / Claude
- 🔧 **内置API** — 开通会员后无需配置，开箱即用
- 🌙 **明暗主题** — 自动适配 X/Twitter 的两种主题
- 🌐 **中英双语** — 自动检测语言切换
- 📝 **自定义视角** — 创建属于你自己的独特视角
- ⚡ **额外次数购买** — 次数不够可单独购买

## 📦 安装

### 方式1：Chrome Web Store（推荐）
> 即将上架，搜索 "X Perspective Switcher" 即可安装

### 方式2：手动安装
1. 下载最新的 [Release ZIP](../../releases)
2. 解压到任意文件夹
3. 打开 Chrome，地址栏输入 `chrome://extensions/`
4. 开启右上角 **"开发者模式"**
5. 点击 **"加载已解压的扩展程序"**
6. 选择解压后的文件夹

## 🚀 快速开始

### 1. 配置 API Key

**免费方案（推荐新手）：**
- 注册 [硅基流动](https://siliconflow.cn)，注册即送 14 元额度
- 在插件设置中填入 API Key，选择免费模型即可使用

**其他模型：**
- OpenAI：[platform.openai.com](https://platform.openai.com)
- DeepSeek：[platform.deepseek.com](https://platform.deepseek.com)
- Claude：[console.anthropic.com](https://console.anthropic.com)

**内置API（会员专属）：**
- 开通会员后，在设置中开启"内置API"开关
- 无需配置，直接使用

### 2. 开始使用

1. 打开 [X/Twitter](https://x.com)
2. 浏览帖子，点击帖子旁边的 **"视角"** 按钮
3. 选择你想要的视角和人设
4. 选择生成模式（反向解读 / 直接回复 / 相似话题）
5. 点击生成，一键复制结果

## 🛠 技术栈

- **Chrome Extension** Manifest V3
- **Vanilla JavaScript**（无框架依赖）
- **Cloudflare Workers** + KV（后端会员系统）
- **CSS3** 渐变动画 + 明暗主题
