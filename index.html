Privacy Policy

X Perspective Switcher

1. Data Collection

This extension only collects the following necessary information:

  • AI API Key: User-configured API keys for OpenAI/DeepSeek/etc., stored locally in the browser only
  • Usage Records: Generation history, membership status, usage count, stored locally in the browser only
  • Custom Perspectives: User-created custom perspective configurations, stored locally in the browser only

2. Data Storage

Important: All data is stored locally in the user's browser and is not uploaded to any server.

We use Chrome's storage API to save data locally on your device:

  • chrome.storage.local - For storing configuration and history
  • chrome.storage.session - For temporary storage of sensitive information

3. Data Transmission

The extension only communicates with external services in the following cases:

  • AI Content Generation: Selected X post content is sent to the user-configured AI service provider (OpenAI, DeepSeek, Anthropic, SiliconFlow, etc.)
  • Redemption Code Verification: Membership redemption codes are verified with our validation server

4. Data Security

We take the following measures to protect your data:

  • API keys are stored in session storage and automatically cleared when the browser is closed
  • All local data is only accessible by the extension itself
  • Your X account information is never shared with any third parties

5. User Rights

You have the following rights:

  • Clear extension data at any time (via "Clear data" in extension management)
  • Uninstall the extension at any time, all local data will be deleted
  • Export your generation history (in the Settings page "History" tab)

6. Third-Party Services

This extension integrates the following third-party AI services. When using these services, you must comply with their privacy policies:

  • OpenAI (https://openai.com/privacy)
  • DeepSeek (https://www.deepseek.com/privacy)
  • Anthropic (https://www.anthropic.com/privacy)
  • SiliconFlow (https://siliconflow.cn/privacy)

7. Privacy Policy Updates

We may update this privacy policy from time to time. Updated policies will be notified to users within the extension.
